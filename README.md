# Тема работы: анализ и прогнозирование стоимости домов в Майами на основе модели множественной линейной регрессии. 
Выбранный для анализа датасет содержит записи о домах, проданных в Майами в 2016 году. 
В работе осуществлены анализ и визуализация данных, анализ влияния факторов на результат, построение модели множевственной линейной регрессии, выбор наилучшей спецификации путем отсева признаков-факторов, тестирование адекватности модели согласно общей схеме, проведение тестов Колмогорова-Смирнова и Глейзера. Получена итоговая модель, содержащая 6 признаков-факторов и коэффициент детерминации 0.72, что говорит о неплохом качестве модели. 
Кроме того, создана интерактивная карта домов с помощью библиотеки Folium.
